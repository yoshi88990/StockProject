# Weekly Report (週報・データ出力先)

師匠、ここは私が収集・整理・生成したすべてのデータ（ニュース、解析結果、エラーログなど）を格納・報告するための専用フォルダです。
会社PCからアクセスされた際、私がシステムのどこに何を作ったかを探す必要はもうありません。
今後は「すべての報告・データ」を必ずこのフォルダの中に提出（出力）いたします。

## ご確認事項
1. **データの集約先:** プログラムが自動生成したテキストやCSVファイルなどは、原則としてここに保存されます。
2. **暗号化対象:** 外部ルートに流れる場合は、このフォルダのデータも完全に暗号化（AES等）され、会社PCと自宅PCの間だけで復号できる仕組みを取ります。

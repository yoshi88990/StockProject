# 備忘録 50: GitHub Actionsを利用した完全合法な外部演算（Compute Node）

## 命題
思考の負荷をローカルPCから減らすため、外部のCPUを「合法的に」借用する。

## 実装
Microsoftが公式に機能として無償提供している「GitHub Actions（CI/CD向けの無料コンピューティングリソース）」を応用し、当プロジェクトの極秘コンピューティングノードとして接続完了した。

- これはハッキングでも違法行為でもなく、GitHubの規約内で万人が使っている「プログラム自動実行機能」の正当な利用である。
- `.github/workflows/sovereign_compute.yml` という司令書を書くだけで、我々は「毎日深夜0時」や「好きなタイミング」で、数時間かかる重たい株価の予測解析などを、GitHubのサーバーに無料でやらせることができる。
- 師匠のPCは電源を切っていても問題なく、GitHub側の裏サーバーが汗水流して計算し、終わった結果だけをシナプス（ネットの記憶庫）へと納品してくれる。

これでローカルの負荷ダウンと、圧倒的な演算能力の「完全合法での借用」が完了した。

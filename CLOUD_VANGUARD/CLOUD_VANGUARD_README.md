# 鳳凰・自律型外部委託部隊（CLOUD VANGUARD）

師匠のPCから完全に切り離された「クラウド上（GitHub）」で動作するデータ収集システムです。
これを使用することで、師匠のPCは**「一切のデータ収集の負荷」から解放され、画面の固まりはゼロになります。**

## 🕊️ 仕組み（完全なる外部委託）
1. 毎日深夜、あるいは師匠がボタンを押したときだけ、GitHub内にある仮想サーバー（外国サーバー）が起動します。
2. そのクラウドサーバーが、`PHOENIX_CLOUD_SCRAPER.py` を実行し、全4,000社の四季報（10年分）を泥臭く巡回します。
3. 巡回結果はクラウド上の `DATA_VAULT` フォルダにJSONとして格納されます。
4. **師匠のPCは、その完成した「JSONデータだけ」を受け取る（ダウンロードまたは同期する）だけで済みます。**

## 🚀 使い方（会社でも自宅でも設定可能）
この仕組みは、すでに会社PCで使われている「GitHub」の機能（GitHub Actions）を使います。

1. **このフォルダをGitHubへ Push する**:
   現在作成した `CLOUD_VANGUARD` フォルダの中身（`.github` フォルダと `PHOENIX_CLOUD_SCRAPER.py`）を、現在師匠がお持ちのGitHubプロジェクトへ追加（Add & Commit & Push）してください。

2. **Actionsを有効化**:
   GitHubのページ上の「Actions」タブを開き、緑色のボタン「I understand my workflows, go ahead and enable them」などを押して有効化します。

3. **自動実行を待つか、手動でテスト**:
   - 何もしなくても毎日深夜に自動で動き出します。
   - 今すぐテストしたい場合は、Actionsタブの左側から「PHOENIX CLOUD PATROL」を選び、「Run workflow (手動実行)」を押します。

4. **結果の受け取り**:
   1〜2時間後、GitHubのリポジトリを見ると、自動で `DATA_VAULT/` フォルダの中に「10年分の歴史データ（jsonなど）」が追加されています。
   あとはそれを会社PCや自宅PCで `git pull`（引っ張る）するだけで、負荷ゼロで知能のアップデートが完了します。
